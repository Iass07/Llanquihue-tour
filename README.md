# LlanquihueTourApp

## Descripción
Aplicación Java que lee información de tours desde un archivo de texto, crea objetos Tour y los almacena en un ArrayList para posteriormente mostrarlos y filtrarlos.

## Estructura

src/
├── model/
│ └── Tour.java
├── data/
│ └── GestorDatos.java
└── ui/
└── Main.java

resources/
└── tours.txt

## Ejecución

1. Abrir el proyecto en IntelliJ IDEA.
2. Verificar que exista el archivo resources/tours.txt.
3. Ejecutar la clase Main.
4. Se mostrará la lista de tours y el filtrado por precio.
